⚡ Power Automate — Fully Automatic Daily Upload

Set up a Power Automate flow that runs at 7am each weekday, reads today's sheet from your Excel file, and posts it to the server automatically — no manual steps.

⚠️ One-time setup required first: Paste your timetable at least once manually using the Timetable tab. This saves the colour → person mapping so the automated flow knows who is who.

Step 2 — Build the flow in Power Automate
1
Go to make.powerautomate.com and sign in with your school Microsoft account.
2
Click Create → Scheduled cloud flow. Set it to run at 7:00 AM, repeat every 1 day.
3
Add a Condition action. Set it to check: dayOfWeek(utcNow(), 'AUS Eastern Standard Time') is not equal to 0 and not equal to 6. This skips weekends.
4
In the Yes branch, add action: Excel Online (Business) → Get a row. Connect to your SharePoint file. Under Sheet Name, use an expression to calculate today's sheet — see Step 3 below.
5
Add action: Excel Online (Business) → List rows present in a table — OR use Get worksheets to read the entire sheet range. Select your file and today's calculated sheet name.
6
Add action: HTTP. Method: POST. URI: the URL above. Add the headers above. Body: use the expression from Step 4.
7
Save and test the flow. Check the server responds with {"ok": true}.
Step 3 — Sheet name expression (Day 1–10)

Use this expression in Power Automate for the sheet name. It calculates the correct Day number based on the cycle start date.

concat('Day ', add(mod(sub(div(sub(ticks(convertFromUtc(utcNow(),'AUS Eastern Standard Time','yyyy-MM-dd')), ticks('2026-02-23')), 864000000000), 1), 10), 1))
⚠ This expression counts weekdays — paste it into a Power Automate "Compose" action first and verify the output matches today's Day number shown above.
Step 4 — Body expression

The HTTP action body should pass each row's values and the row index. Use this template and replace outputs('List_rows')?['body/value'] with the actual output from your List rows action.

@{ "rows": outputs('List_rows')?['body/value'] }
📌 Important: Power Automate's Excel connector does not return cell background colours. The server will match columns by position using the colour layout saved from your last manual paste. If you change the spreadsheet layout, do one manual paste to update the mapping.
